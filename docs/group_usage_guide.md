# 群组功能使用指南

## 📱 第一步：将机器人添加到群组

### 方法1：通过机器人添加

1. 在 Telegram 中打开你的机器人
2. 点击机器人资料页的菜单按钮(...)
3. 选择"添加到群组"或"Add to Group"
4. 选择你想添加的群组
5. 点击"添加"确认

### 方法2：通过群组添加

1. 打开你的 Telegram 群组
2. 点击群组名称进入群组信息页
3. 点击"添加成员"或"Add Members"
4. 搜索你的机器人名称（例如：@your_bot_name）
5. 选择并添加

### ⚠️ 重要提示

- 确保机器人有**发送消息**的权限
- 建议给机器人**管理员权限**（可选，但更流畅）
- 如果是私密群组，需要先添加机器人才能使用

## 🔮 功能2：个人塔罗占卜（自动进排行榜）

### 使用方法

在群里发送：
```
/tarot 我今天运势如何？
```

或者：
```
/tarot 我应该换工作吗？
```

### 完整流程

1. 发送命令后，会出现"🎴 抽牌"按钮
2. 点击按钮开始抽牌
3. 看洗牌动画（4个阶段）
4. 显示精简版结果
5. 点击"📜 详细解读"查看完整分析

## ❓ 常见问题

### Q1: 机器人没有回应？

**解决方法**：
- 检查机器人是否有发送消息权限
- 确认命令格式正确（有 / 符号）
- 查看机器人是否在线（私聊发送 /start 测试）

### Q2: 如何让机器人只响应特定命令？

**答**：机器人默认只响应命令，不会主动发言，不会影响正常聊天。

### Q3: 排行榜什么时候重置？

**答**：每天零点自动重置，新的一天重新排名。

### Q4: 可以和多个人同时PK吗？

**答**：暂时只支持1v1，多人对战在后续版本中实现。

### Q5: 占卜结果会被其他群看到吗？

**答**：不会，每个群的数据完全独立，互不影响。

### Q6: 可以看历史排行榜吗？

**答**：目前只显示当天的，历史数据功能在规划中。

### Q7: PK被拒绝了怎么办？

**答**：换个人挑战，或者改天再约 😄
